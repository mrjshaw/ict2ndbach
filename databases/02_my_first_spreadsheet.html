My First Spreadsheet

Please create this attendance sheet using Google Sheets or Excel and upload the document or a screenshot of the document to your portfolio under databases.

  • The first name in the list must be yours
  • There must be 9 names in the list
  • The dates must be for this year and 4 days apart
  • There must be one absence marked with a dash (-)

*hint, you can inseert the date using the hotkey ctrl+; in excel.

Advanced Instructions

Adding an Image to GitHub Pages

Do the task as normal but adding it to GitHub Pages is going to be more complicated than Wix

First, create a folder named "img" in the databases folder.

Second, save a screenshot (png or jpg) of your spreadsheet in that img folder.

Last, go to your databases/index.html file and add the heading "My First Spreadsheet" and the image using the img tag
